# Reedly

Reedly is a lightweight social app prototype for book readers, inspired by Twitter.

## Features

- Share what you read and post book reviews
- Repost button (`🔁`) similar to a retweet
- Book-style like button (`📖`) instead of a heart
- Optional video URL on a post for book video content
- Quick links to TikTok and Instagram reader communities

## Run

```bash
npm install
npm start
```

Then open `http://localhost:3000`.

## Test

```bash
npm test
```
